# Інструкція: Налаштування маппінгу екіпірування

## Що вже зроблено ✅

1. ✅ Файли екіпірування скопійовані в `public/equipment/`
2. ✅ Система відображення налаштована
3. ✅ Структура коду готова

## Що потрібно зробити

### Крок 1: Визначити ID файлів для предметів

1. Відкрийте папку: `public/equipment/front/0-male/`
2. Подивіться на файли (0.png, 1.png, 100.png...)
3. Визначте, який файл відповідає якому предмету

**Приклад:**
- `0.png` - може бути базовим шлемом NG
- `1.png` - може бути базовою бронею NG
- `100.png` - може бути D-grade шлемом

### Крок 2: Оновити маппінг

Відкрийте файл: `src/utils/equipment/itemEquipmentMapping.ts`

Оновіть `ITEM_EQUIPMENT_MAPPING`:

```typescript
export const ITEM_EQUIPMENT_MAPPING: Record<string, number> = {
  // NG ARMOR
  ng_helmet_leather: 0,    // Замініть на правильний ID
  ng_armor_leather: 1,     // Замініть на правильний ID
  ng_gaiters_leather: 2,   // Замініть на правильний ID
  // ... і так далі
};
```

### Крок 3: Перевірити

1. Запустіть гру
2. Одягніть предмет
3. Перевірте, чи відображається екіпірування на моделі

## Поради

- Почніть з невеликої кількості предметів (наприклад, тільки NG)
- Використовуйте браузерну консоль для перевірки шляхів
- Якщо зображення не відображається, перевірте:
  - Чи правильний ID в маппінгу
  - Чи існує файл за цим шляхом
  - Чи правильна раса/стать

## Автоматизація (майбутнє)

Можна створити скрипт, який автоматично знаходить відповідність між
назвами файлів та предметами, але для початку краще зробити вручну.
